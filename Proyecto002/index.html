Cookies

¿Qué es una cookie?

Casi cualquier página web que utilices maneja este formato, no es más que un fichero de texto que se almacena en su navegador,es decir, es un archivo creado por un sitio web que contiene pequeñas cantidades de datos y que se envían entre un emisor y un receptor (El emisor sería el servidor/host y el receptor es el navegador que se está utilizando).

Cuando se visita una página web por primera vez, se guarda una cookie en el navegador con un poco de información. Luego, cuando se visita nuevamente la misma página, el servidor pide la misma cookie para arreglar la configuración del sitio y hacer la visita del usuario tan personalizada como sea posible.
